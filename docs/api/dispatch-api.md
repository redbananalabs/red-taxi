# Dispatch API

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /bookings/{id}/allocate | Allocate driver |
| POST | /bookings/{id}/complete | Mark complete |
| POST | /dispatch/offer | Offer job to driver |
| POST | /dispatch/accept | Accept job offer |
| POST | /dispatch/reject | Reject job offer |
