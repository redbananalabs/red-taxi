# Dispatch Workflow

## Manual Dispatch

1. View unallocated jobs
2. View available drivers
3. Match job to suitable driver
4. Drag to scheduler OR click allocate
5. Driver receives notification
6. Job status → Allocated

## Auto Dispatch

1. Job confirmed
2. System finds best driver
3. Offer sent to driver app
4. Driver accepts/rejects
5. If rejected, offer to next
6. Escalate if no acceptance
