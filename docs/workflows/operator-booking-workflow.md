# Operator Booking Workflow

## Steps

1. Call comes in / request received
2. Operator opens booking form
3. Enter pickup address (autocomplete)
4. Enter destination address
5. System calculates quote
6. Enter passenger details
7. Select vehicle type if needed
8. Press END to submit
9. Confirmation sent to passenger
10. Job appears on scheduler

## Time Target
Complete booking in < 30 seconds
