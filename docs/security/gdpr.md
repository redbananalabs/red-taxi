# GDPR Compliance

## Data Subject Rights

- Right to access
- Right to rectification
- Right to erasure
- Right to portability

## Data Retention

| Data Type | Retention |
|-----------|-----------|
| Booking data | 7 years (tax) |
| Location history | 90 days |
| Messages | 2 years |
| Deleted accounts | 90 days then purge |
